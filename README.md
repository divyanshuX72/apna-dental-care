# 🦷 Apna Dental Clinic Website

A premium, responsive, and modern dental clinic website developed for **Apna Dental Clinic** using **HTML, CSS, and JavaScript**.

The website is designed with a clean healthcare-focused UI, smooth animations, WhatsApp appointment booking, Google Reviews integration, and mobile-friendly responsive design.

---

# ✨ Features

- Premium Healthcare UI Design
- Fully Responsive Layout
- Google Reviews Integration
- WhatsApp Appointment Booking
- Modern Services Section
- Clinic Gallery Page
- About Doctor Section
- Contact & Google Maps Integration
- Smooth Animations
- SEO Friendly Structure
- Fast Loading Performance
- Mobile Optimized Design
